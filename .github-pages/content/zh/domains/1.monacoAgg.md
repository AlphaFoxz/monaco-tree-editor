# Monaco聚合

- 拓展了monaco-editor的功能

## 使用方法

```vue
<script setup lang="ts">
import { useMonaco, Editor as MonacoTreeEditor } from 'monaco-tree-editor'
import * as monaco from 'monaco-editor'

const monacoAgg = useMonaco(monaco)
monacoAgg.event.
</script>

<template>
  <MonacoTreeEditor />
</template>
```
